# RPS App V2

A Flutter e-commerce furniture application with multi-role support (User, Employee, Admin).

## Features

### User Role
- User authentication (login/register)
- Product catalog browsing
- Shopping cart and order flow
- Payment verification via slip upload
- Order status tracking
- Product reviews and ratings

### Employee Role
- View order details
- Track and update order status
- Update delivery/shipping status

### Admin Role
- Approve customer orders
- Monitor order statuses
- View sales reports (revenue, profit/loss)
- Manage product inventory (add, update, delete products)

### General Features
- Localization support
- Image picking capability

## Getting Started

### Prerequisites
- Flutter SDK: ^3.7.0
- Dart

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Project Structure
- `/lib` - Main application code
- `/assets` - Images and resources
- `/android` - Android-specific code
- `/ios` - iOS-specific code
